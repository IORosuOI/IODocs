# docsIO - Web Programming (Semester III)

This directory contains iterative frontend implementations for the **Web Programming** course at UBB. 

### 🛠️ Development Context
The code follows a progressive structure, adhering to strict (and sometimes legacy) laboratory constraints to demonstrate an understanding of web evolution.

- **Lab 1:** Pure HTML structure (HTML 4.01 Transitional & HTML5).
- **Constraints:** No CSS; table-based layout; strict W3C validation; 15+ interactive controls.
- **Goal:** Core UI for document management (Dashboard & Settings).
